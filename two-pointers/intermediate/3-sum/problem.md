# 3Sum Problem

## Problem Statement

Given an integer array `nums`, return all unique triplets `[nums[i], nums[j], nums[k]]` such that:

- `i ≠ j`, `i ≠ k`, and `j ≠ k`
- `nums[i] + nums[j] + nums[k] == 0`

The order of triplets in the output does not matter.

### Constraints:

- `3 ≤ nums.length ≤ 500`
- `-10^3 ≤ nums[i] ≤ 10^3`
